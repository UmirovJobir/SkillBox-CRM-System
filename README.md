## Описание проекта

CRM-система для отдела продаж. Позволяет:
- Вести **каталог услуг** компании
- Создавать **рекламные кампании** с бюджетом
- Фиксировать **лидов** (потенциальных клиентов) из рекламы
- Конвертировать лидов в **активных клиентов**
- Оформлять **контракты** на услуги
- Видеть **статистику** эффективности каждой рекламной кампании
- Управлять **правами доступа** для разных сотрудников

### Стек технологий

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/UmirovJobir/SkillBox-CRM-System.git
cd SkillBox-CRM-System
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Применить миграции

```bash
python manage.py migrate
```

### 4. Создать суперпользователя

```bash
python manage.py createsuperuser
```

Или использовать готового (если база уже заполнена):
- Логин: `admin`
- Пароль: `admin123`

### 5. (Опционально) Заполнить тестовыми данными

```bash
# Добавить тестовые данные
python manage.py seed

# Очистить и заполнить заново
python manage.py seed --clear
```

Команда создаёт:
- 8 услуг
- 6 рекламных кампаний
- 30 лидов
- ~12 активных клиентов
- ~20 контрактов

### 6. Запустить сервер

```bash
python manage.py runserver
```

Открыть в браузере: `http://127.0.0.1:8000/`

## Обработка нестандартных ситуаций

### Попытка открыть несуществующую запись

```
GET /products/9999/
→ 404 Not Found
```

### Доступ без авторизации

```
GET /products/
→ 302 Redirect → /accounts/login/?next=/products/
```

### Доступ без нужного права

```
GET /products/  (залогинен, но нет права view_product)
→ 403 Forbidden
```

### Отправка пустой формы

```
POST /products/new/  (пустое поле "Название")
→ 200 OK — форма показывается снова с ошибкой:
   "Обязательное поле."
```

### Создание клиента из уже использованного лида

```
POST /customers/new/  (лид уже привязан к другому клиенту)
→ 200 OK — форма с ошибкой:
   "Активный клиент с таким полем Лид уже существует."
```

## Управление правами доступа

Права назначаются через Django Admin: `http://127.0.0.1:8000/admin/`

Доступные права для каждого раздела:

| Право | Доступ |
|-------|--------|
| `products.view_product` | Просмотр списка и деталей услуг |
| `products.add_product` | Создание услуги |
| `products.change_product` | Редактирование услуги |
| `products.delete_product` | Удаление услуги |
| `advertisements.view_advertisement` | Просмотр реклам и статистики |
| `leads.view_lead` | Просмотр лидов |
| `customers.view_customer` | Просмотр клиентов |
| `contracts.view_contract` | Просмотр контрактов |

Боковое меню автоматически скрывает разделы, к которым у пользователя нет доступа.
